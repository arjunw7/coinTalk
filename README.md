# coinTalk
Single-page news and blogging application focused in cryptocurrencies, and blockchain technology.

## Technologies
- **Language**: JavaScript
- **Frontend**: React
- **Backend**: Node, Express
- **Databases**: PostgreSQL, AWS S3
